## homebrew-git-devbox

### install it

```bash
brew tap viqueen/git-devbox
brew install git-devbox
```

### package info

* source : [codenav](https://github.com/viqueen/git-devbox)

### update on new releases

* run the following
```bash
node upgrade.js
```

* commit the updates to `git-devbox.rb` file
